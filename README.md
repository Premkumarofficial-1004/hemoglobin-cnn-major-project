# Hemoglobin Estimation using CNN

## Project Overview
This project aims to estimate hemoglobin levels using image processing and deep learning techniques. A Convolutional Neural Network (CNN) model is trained on fingertip images to predict hemoglobin concentration non-invasively.

## Technologies Used
- Python
- TensorFlow / PyTorch
- OpenCV
- NumPy
- Scikit-learn

## Project Structure
dataset/ – Training images  
models/ – Saved CNN model  
src/ – Helper scripts  
train_model.py – Model training code  
requirements.txt – Required Python libraries  

## Objective
To develop a non-invasive hemoglobin estimation system using computer vision and deep learning.
