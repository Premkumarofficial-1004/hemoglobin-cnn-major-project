Non-Invasive Hemoglobin Estimation using CNN

📌 Project Description

This project presents a non-invasive approach to estimate hemoglobin levels using image processing and deep learning techniques. A Convolutional Neural Network (CNN) is trained on fingertip images to predict hemoglobin concentration without the need for blood samples.

🧠 Technologies Used

- Python
- TensorFlow / Keras
- OpenCV
- NumPy
- Streamlit

⚙️ Features

- Image-based hemoglobin prediction
- CNN model for classification/regression
- User-friendly Streamlit interface
- Non-invasive and fast results

▶️ How to Run

1. Install dependencies:

pip install -r requirements.txt

2. Run the app:

streamlit run src/app.py

⚠️ Note

Due to GitHub file size limits, the trained model file (.h5) is not included in this repository.
